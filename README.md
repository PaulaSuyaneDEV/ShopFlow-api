
🛒 ShopFlow API

API REST de e-commerce desenvolvida com Spring Boot, com foco em boas práticas, testes e modelagem de domínio.

🚀 Tecnologias
Java 17
Spring Boot
Spring Data JPA
PostgreSQL
Swagger
JUnit + Mockito

📦 Funcionalidades
CRUD de produtos
criação de pedidos
cálculo automático de total
validação de regras de negócio
tratamento de erros
testes automatizados

▶️ Como rodar o projeto
git clone <seu-repo>
cd shopflow
mvn clean install
mvn spring-boot:run

🌐 Acessar a API
http://localhost:8080
📄 Swagger (Documentação da API)

http://localhost:8080/swagger-ui/index.html

📌 Exemplos de endpoints
Criar produto
POST /products
Listar produtos
GET /products
Criar pedido
POST /orders
